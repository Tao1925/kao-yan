Chap1 教育学与教师专业化

一、当代教育学的发展趋势是
	1、学科性质的多元和融合
		教育学 -> 经验科学，哲学 -> 融合
	2、学科内容发展的分化与综合
		教育学的趋势 -> 分化为学科群
		内部分化 -> 小/中/高，职业/特殊/师范，德育论/课程论/教学论
		与其他学科结合 -> 教育哲学/社会学/心理学...
	3、研究基础、模式和方法的多元化
		基础 -> 广泛的学科领域，社会学/经济学/哲学...
		模式和方法 -> 实验科学的方法/思辨哲学的方法
	4、理论研究与实践改革的关系日益密切
	
二、教师专业化的概念是
	教师在整个职业生涯发展中，
	通过专门的培养制度和管理制度 -> 成长条件
	使他们具有独特的职业要求和职业条件 -> 成长结果
	的成长过程
	不是一种终结性概念，而是发展性概念
	
三、教育学在教师的专业化中有哪些独特价值
	1、教育学素养是教师专业的重要基石之一
		教师具有“双专业性” -> 学科专业教育/教育专业教育
		教育学知识是专业化教师不可缺少的专业知识结构的重要组成部分
	2、教育学有助于提升教育工作者的教育自觉性
		学习教育学理论有助于帮助教师缩短从不成熟到成熟的路程
		优秀的教师依靠的是高度的自觉性，只有具有高度自觉性的教师才能领悟教育的真谛
	3、教育学能为反思型、研究型教师打下基础
		教育与研究交融，教学与反思的结合，能够使教师成为更富理性的行动者
		只有通过教育学的学习，教师才能突破狭隘的个人经验，以一种新的教育观反思和超越自我
	
Chap2 教育、学校和学制

一、制度化教育的形成过程是
	1、非形式化教育 -> 教育与生活、生产浑然一体
	2、形式化教育 -> 出现教育机构，但缺乏明确的制度
	3、制度化教育 -> 学校教育制度基本建立

Chap3 社会、人、教育

一、个体发展对教育的制约体现在
	1、个体发展水平对教育的制约
		先天因素 -> 遗传素质/身心成熟机制
		后天发展的水平 -> 知识经验/情感意志 -> 影响教育的选择/个体发展的自觉性
	2、个体身心发展的规律对教育的制约
		阶段性/顺序性/差异性/不平衡性/整体性

二、马克思人类社会发展的三种形态说
	1、古代社会（原始/封建/奴隶社会）
		人的发展状态：人的依附关系 教育形态：依附性关系的训练
	2、近现代社会（资本/社会主义社会）
		人的发展状态：以物的依赖性为基础 教育形态：个人主体教育
	3、当代和未来社会（共产主义社会）
		人的发展状态：建立在个人全面发展基础上的自由个性 教育形态：自由个性的教育
	
三、教育至于社会的适应性和超越性体现在
	教育是社会政治经济和生产力发展的结果，与其相适应
	现代社会解放了人，使其成为社会的适应者和创造者
	现代教育就是要培养改造现实世界的社会历史活动的主体，实现教育的超越性
	教育社会功能的实现，推动着社会生产力和政治经济制度的变革， 这也是超越性的表现

Chap4 教育目的

一、教育目的确定的依据有
	1、社会历史发展的客观要求 -> 生产力水平/文化发展需要/社会政治经济制度
	2、受教育者身心发展的规律 -> 教育目的是针对对象而言，人的发展是根本/制定需考虑身心发展特点
	3、制定教育目的者的价值取向和教育理念

二、中国学生发展核心素养有
	1、文化基础 -> 人文底蕴/科学精神
	2、社会参与 -> 责任担当/实践创新
	3、自主发展 -> 学会学习/健康生活
	
三、布鲁姆教育目标分类学的内容是
	将教育目标分为认知领域、情感领域、动作技能领域
	
四、我国的教育目的实施中存在哪些问题
	1、全面发展和因材施教
	2、应试教育和素质教育
	3、理想性和现实性
